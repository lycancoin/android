Quelques mots sur la sécurité de vos Lycancoins:

  • Lycancoins dans le Lycancoin Wallet sont stockés dans le téléphone. Si vous le perdez, vous perdez vos Lycancoins. Comme de l'argent liquide dans votre porte-feuille.
  • Avant de désinstaller Lycancoin Wallet (ou d'effacer les données de l'application), transferez vos Lycancoins dans un autre wallet. Tout lycancoins restant seront perdus.
  • Les transactions sont irréversibles. Si vous envoyez vos Lycancoins dans le vide,il n'y a aucun moyen de les récupérer sauf si vous connaissez le destinataire et qu'il est d'accord pour les renvoyer.
  • Gardez votre téléphone Android en sécurité! N'installez que des applications de confiance. Des applications pirates pourraient essayer de voler votre wallet.
  • Minimisez les risques! Utilisez Android Wallet seulement pour des petits montants pour un usage quotidien, et gardez vos économies en sûreté.
